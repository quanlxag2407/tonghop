Code Html
html table các thuộc tính trong table(border,align,width,bgcolor)
html img các thuộc tính trong img(src)
html a(tạo liên kết) các thuộc tính trong thẻ a(href,name,target)
html link(nhúng nguồn link bên ngoài) các thuộc tính trong thẻ Link(href,rel="stylesheet",type,media)
html ul và or các thuộc tính trong thẻ ul và ol(type), thẻ "li"được lòng vào thẻ ul và ol
html audio các thẻ trong audio(controls="1",loop,mutead)
html video các thẻ trong video(autoplay,controls,loop,muted,poster,height,width)
